# AI-Powered SEO Content Production for B2B SaaS

This repository contains structured research on AI-powered SEO content
production for B2B SaaS companies.

The project studies 10 practitioners who actively work in SaaS SEO,
content strategy, AI-assisted content production, organic growth,
and content operations.

## Research Objectives

The objectives of this project are to:

- Identify high-signal practitioners in AI-powered SEO and B2B SaaS content.
- Collect recent LinkedIn posts, YouTube transcripts, articles, podcasts,
  and other relevant materials.
- Organize the research into a structured GitHub repository.
- Use APIs and technical tools to automate parts of the research process.
- Build a reliable research foundation for a future practical playbook.
