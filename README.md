# AI-Powered SEO Content Production for B2B SaaS

This repository contains structured research on AI-powered SEO content production for B2B SaaS companies.

The project studies 10 practitioners who actively work in SaaS SEO, content strategy, AI-assisted content production, organic growth, content operations, and AI search visibility.

## Research Objectives

The objectives of this project are to:

* Identify high-signal practitioners in AI-powered SEO and B2B SaaS content.
* Collect recent LinkedIn posts, YouTube transcripts, articles, podcasts, and other relevant materials.
* Organize the research into a structured GitHub repository.
* Use APIs and technical tools to automate parts of the research process.
* Build a reliable research foundation for a future practical playbook.

## Collection Methods

Research materials are collected using:

* Public LinkedIn posts
* Publicly available YouTube transcripts
* Articles and newsletters
* Podcast episodes and webinars
* Python scripts and APIs
* Manual validation and annotation

Where automated collection is not technically or legally practical, materials are collected manually from publicly accessible sources.

## Technical Implementation

The repository includes a Python script that:

* Accepts a YouTube URL or video ID
* Retrieves publicly available transcripts
* Extracts transcript metadata
* Groups transcript segments into readable sections
* Saves the results as structured Markdown files

The current implementation uses the `youtube-transcript-api` Python package.

## Research Ethics

This repository is intended for research and educational purposes.

* All original creators are credited.
* Source URLs and publication dates are preserved.
* Full copyrighted articles are not reproduced.
* LinkedIn content is collected manually from publicly accessible posts.
* YouTube transcripts are collected only where transcripts are publicly available.
* Research notes include summaries and annotations rather than presenting third-party work as original content.
